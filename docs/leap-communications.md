# LEAP — Communications & Templates

Templates are infrastructure — not a separate module. Managed centrally in LEAP Admin. Used across all modules.

**Template types:**
- Email templates
- Document templates (merge fields → PDF)
- Application templates
- E-signature workflows
- Automated notifications

**Merge fields** pull from any related object in the database — property name, project amount, state, program, contact name, dates, status, assigned technician, etc.

**Templates are program and state specific.** Wisconsin multifamily IRA HOMES heat pump communications are different from North Carolina single-family HEAR communications. Templates are organized by state, program, project type, and record type.

**Every sent communication is logged** as a completed activity against the record it relates to. Full communication history visible on every property, project, and contact timeline.

**Automated triggers** — status changes trigger specific templates automatically. Manual sending available from any record at any time.
