# LEAP — Vehicles & Fleet

Every vehicle activity captures:
- Odometer reading
- GPS coordinates
- Timestamp
- Named individual responsible
- Activity type

**Activity types:** Pre-trip inspection, post-trip inspection, fuel log, maintenance, damage report, mileage log.

**Vehicle issuance:** To the Team Lead by name. Team Lead is the responsible driver for that vehicle for that day. Issued by Shop Steward, authorized by Director of Field Services or Project Coordinator based on the day's schedule.

**Morning issuance to Team Lead:**
- Vehicle keys
- Crew phone(s)
- Job kit(s)

Each issuance is a separate record with a named responsible party.

**Crew phone issuance logic:** Driven by work type. Work types that require individual work order completion issue a crew phone to each technician. Work types that require only team-level completion issue only one crew phone to the Team Lead.
