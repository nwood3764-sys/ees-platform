# LEAP — Property Hierarchy

The property hierarchy is the spine of the entire LEAP database. Every work order, project, rebate, income qualification, technician assignment, and photo ultimately traces back to a Unit or a Building.

```
Property Owner
  └── Property Management Company
        └── Property
              └── Building
                    └── Unit
```

All of our programs serve low-income properties. Affordability categories include:
- Section 8 / HUD
- Low-Income Housing Tax Credit (LIHTC)
- Naturally Occurring Affordable Housing (NOAH)
- Disadvantaged Community (DAC)
- NEST Community
- Other program-specific designations

A property can qualify for multiple programs simultaneously. Program strategy is property-specific — never one size fits all.
