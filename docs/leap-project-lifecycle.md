# LEAP — Project Lifecycle

Every project follows this lifecycle. Status names are explicit. Each status has a named owner and implies a next action.

## Stage 1 — Property Identification & Research
Property discovery, building inventory, unit count, building characteristics, subsidy type, affordability category classification.

## Stage 2 — Outreach & Relationship Mapping
Decision-maker identification at national, regional, state, community, and property level. Opportunity Contact Roles track multiple stakeholders and their role in the decision-making process per property.

## Stage 3 — Enrollment & Income Qualification
HAF agreements, HUD agreements, income qualification statements, rent rolls, census tract verification, DAC/NEST community designation. Tracked at building and unit level depending on program requirements.

## Stage 4 — Site Assessment & Energy Audit
ASHRAE Level 2 assessment, photos, 3D Matterport captures, building measure characteristics. Snug Pro for single-family. Asset Score for multifamily. Baseline and improvement package modeling. OpenStudio export for energy usage data. Audit findings reviewed and verified before proceeding.

## Stage 5 — Incentive Applications
Program-specific applications submitted per program per project. Multiple simultaneous applications on one property are common — IRA HOMES, IRA HEAR, Focus on Energy insulation, Focus on Energy Midstream, QEP, DAC, utility-specific incentives. Each application has its own status lifecycle and its own owner.

## Stage 6 — Project Reservation
Issued by the program after application approval. Triggers project scheduling. One project can have multiple reservations from multiple programs.

## Stage 7 — Project Planning
Work plans created. Work orders built per building or unit. Work steps defined. Materials and equipment lists generated from bill of materials. Shop kitting work orders created. Crew assignments made. Schedule built.

## Stage 8 — Project Implementation
Field execution. Team Lead executes work orders per work plan. Work steps completed with required evidence. Before and after photos captured. Project Coordinator tracks daily progress. Director of Field Services manages real-time execution.

## Stage 9 — Verification
Project Coordinator reviews completed work orders. Corrections identified and reassigned to Project Site Lead and Team Lead. Remediation completed and resubmitted. Final verification sign-off required before utility inspection.

## Stage 10 — Utility Site Verification
External — program administrator or utility comes on site. Program-specific paperwork and photo submission workflow.

## Stage 11 — Project Payment Request
Submitted per program at project close. Multiple payment requests per project if multiple programs. Processing timelines tracked per program and payment body.

## Stage 12 — Payment Receipt & Close-Out
Incentive payment received and recorded. Property close-out report generated. Internal after-action report completed per building — office and field teams.
